This is for cognizant laptop or any company laptop which is blocking the node js

for this project we need to make some changes in the ui.frontend. We need to do because ui.frontend is blocked by the laptop.

mvn org.apache.maven.plugins:maven-archetype-plugin:3.2.1:generate -B -D archetypeGroupId=com.adobe.aem -D archetypeArtifactId=aem-project-archetype -D archetypeVersion=40 -D artifactId=mysite -D appTitle="mysite" -D appId="mysite" -D groupId="com.adobe.aem.mysite" -D includeDispatcherConfig=n -D aemVersion="6.5.13"

Need to add skip configuration in plugin
<groupId>com.github.eirslett</groupId>
                <artifactId>frontend-maven-plugin</artifactId>
// This configuration should be added to skip ui.frontend
                <configuration>
                    <skip>true</skip>
                </configuration>

<groupId>org.apache.maven.plugins</groupId>
                <artifactId>maven-assembly-plugin</artifactId>
// This configuration should be added to skip ui.frontend

                <configuration>
                    <skipAssembly>true</skipAssembly>
                </configuration>

After that we need to remove ui.frontend dependency in ui.apps
